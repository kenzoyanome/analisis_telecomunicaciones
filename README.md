# analisis_Telefonia
Analisis de comportamiento del negocio con informacion registrada hasta el 2024. Exploracion, limpieza y analisis de datos para construir un perfil estadistico de clientes, detectar comportamientos atípicos y segmentacion de clientes. Identificar patrones de consumo, diseñar estrategias de retención y mejoras en los planes ofrecidos por la empresa.
